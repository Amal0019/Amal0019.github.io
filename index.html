Track your daily calorie and nutrient intake with ease

View on GitHub

📝 Project Overview

This web application allows users to track their daily calorie intake and nutritional breakdown including proteins, carbs, and fats. It supports food logging, chart-based dashboards, weekly/monthly reports, and user authentication.

🚀 Features

  • User authentication (signup/login/logout)
  • Add and remove food items with nutritional values
  • Track calories, proteins, fats, and carbohydrates
  • Interactive dashboard with charts
  • Streaks, goals, and progress tracking
  • Export reports (PDF/CSV)
  • Email notifications (weekly/monthly reports)
  • Mobile-friendly design

🛠️ Technologies Used

  • Frontend: HTML, CSS, Bootstrap
  • Backend: Django (Python)
  • Database: SQLite / MySQL
  • Charts: Chart.js or Recharts
  • Email: Django Email Backend
